# Role-first Scenarios

## Student

### Main Question
What should I do right now?

### Mandatory UI Blocks

- Focus task card
- Active / done / risk KPI
- Quick actions: start / done
- Deadline visibility

### Definition of Done

Student can identify next action in under 10 seconds and update status in one interaction.

## Parent

### Main Question
Where is risk and what changed?

### Mandatory UI Blocks

- Risk today summary
- Overdue and attention counters
- Task list sorted by urgency
- Telegram linked status

### Definition of Done

Parent sees actionable risk items without navigating through student-only details.

## Teacher

### Main Question
How fast can I create and control tasks?

### Mandatory UI Blocks

- Smart task input
- AI structured preview + confidence
- Assign controls (student/parent)
- Status overview and overdue control

### Definition of Done

Teacher can input unstructured text, verify structure, and publish in one page flow.
